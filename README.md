AI Fashion Stylist Pro – A full stack web application that uses machine learning to predict and suggest fashion styles.

Features:
- AI-based prediction system using TensorFlow/Keras
- REST API built with Flask
- Endpoints: /predict, /health
- Deployed on Render
- Frontend integration with backend API

Tech Stack:
Python, Flask, TensorFlow, HTML, CSS, JavaScript

Deployment:
Backend hosted on Render with Gunicorn
Frontend deployed separately

This project demonstrates how machine learning models can be integrated into real-world web applications.
